Aspect Ratio

A containing element allowing to set the aspect ratio for various media such as images and video.

A common use-case for this is when lazy loading images. We do not want layout shifts when loading images. The aspect ratio sets a defined space for the media to load in.

We are utilising both the modern aspect-ratio: 3/4 CSS property, and the traditional "intrinsic ratio" hack which sets padding on an element equal to its aspect ratio. Further information here: https://css-tricks.com/aspect-ratio-boxes/

When to Use

You only need .aspect-ratio for media whose size you want the option to adjust. For example, many sections in our baseline theme offer the merchant a choice of different image sizes and proportions.

Standard images in their natural proportions work fine without this.

How it works​

  • The .aspect-ratio class will set a padding-bottom or --aspect-ratio to declare the size of an element.
  • The child media will be absolutely positioned relative to the aspect-ratio parent

How to use​

<div class="image__wrapper">  <div class="aspect-ratio aspect-ratio--[size modifier]"        style="         padding-bottom: {{ 100.0 | divided_by: image.aspect_ratio }}%;          --aspect-ratio: {{ image.aspect_ratio }}">    <img class="image" [image attributes]>  </div></div>

The structure is generally as follows

  • wrapper element
    • .aspect-ratio, modifier classes, style attribute
      • media

We use the style attribute to set the natural proportions via padding-bottom and an --aspect-ratio variable for modern browsers. If we don't have an .aspect-ratio--[size-modifier], then these natural proportions will be used. Otherwise, the size modifier classes will override.

Examples​

The below example shows an article-item image. We provide the base class .aspect-ratio followed by a sizing modifier. This sizing modifier can often be selected by a customer in the Shopify backend as shown by the {{ image_size }} liquid variable in the example below.

If using natural image proportions, we will use a style attribute with padding, and an --aspect-ratio variable for modern browsers.

<a href="{{ article.url }}" class="article-item__image-wrapper">  <div class="article-item__aspect-ratio aspect-ratio {% if image_size != 'natural' %}aspect-ratio--{{ image_size }} aspect-ratio--cover{% endif %}" style="padding-bottom: {{ 100.0 | divided_by: article.image.aspect_ratio }}%; --aspect-ratio: {{ article.image.aspect_ratio }}">    <img       class="image"        loading="lazy"       sizes="{{ sizes_attribute }}"      {% render 'image-attributes', image: article.image, sizes: '600,700,800,1000,1200,1400,1600,1800,2000' %}>  </div></a>

Options​

Size Modifiers​

ModifierPaddingRatio
.aspect-ratio--square100%1:1
.aspect-ratio--short75%4:3
.aspect-ratio--tall133%3:4
.aspect-ratio--tallest150%2:3
.aspect-ratio--wide56.25%16:9

Other Modifiers​

  • .aspect-ratio--cover: Sets media to fill all available space
